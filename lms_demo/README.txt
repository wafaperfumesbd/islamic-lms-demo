==============================================
  WHITE LABEL ISLAMIC LMS — CLIENT DEMO
==============================================

ব্যবহারের নিয়ম:
-----------------
1. ZIP ফাইলটি Extract করুন
2. "index.html" ফাইলটি যেকোনো Browser-এ Open করুন
   (Chrome, Firefox, Edge — সবগুলোতে কাজ করবে)
3. উপরের সবুজ Navigation bar থেকে পেজগুলো দেখুন

ফাইল তালিকা:
--------------
index.html              — হোমপেজ (সম্পূর্ণ)
courses.html            — কোর্স তালিকা (ফিল্টার সহ)
course-detail.html      — কোর্স বিবরণ (syllabus, enroll)
lesson-player.html      — ভিডিও লেসন প্লেয়ার
student-dashboard.html  — শিক্ষার্থীর ড্যাশবোর্ড
payment.html            — bKash/Nagad পেমেন্ট ফর্ম
certificate.html        — ডিজিটাল সার্টিফিকেট
admin-dashboard.html    — অ্যাডমিন ড্যাশবোর্ড
admin-payments.html     — পেমেন্ট যাচাই প্যানেল
theme-customizer.html   — থিম কাস্টমাইজার (live preview)
homepage-builder.html   — হোমপেজ বিল্ডার (drag-drop)
style.css               — Shared CSS (সব পেজে ব্যবহার)

Interactive Features:
---------------------
- Theme Customizer: রঙ বদলালে live preview আপডেট হয়
- Homepage Builder: blocks drag করে সাজানো যায়, toggle on/off
- Payment page: bKash/Nagad/Bank tab switch করা যায়
- Admin Payments: পেমেন্ট select করে approve/reject করা যায়

Tech Stack (Actual Product):
-----------------------------
Next.js 15 + TypeScript + Tailwind CSS
Supabase (DB + Auth + Storage)
Vercel (Hosting — Free Tier)
bKash/Nagad Manual Payment
